LAS Calculus
Computing Derivatives
Homework 4
  1. Find \(f'(x)\) for each of the following.
    1. \(f(x)=x^7\)
      \(f'(x)=7x^6\).
    2. \(f(x) = \sqrt{x}\)
      \(f'(x)=\frac{1}{2}x^{-\frac{1}{2}}\).
    3. \(f(x) = 1/\sqrt{x}\)
      \(f'(x)=-\frac{1}{2}x^{-\frac{3}{2}}\).
    4. \(f(x) = 2/x^4\)
      \(f'(x)=-8x^{-5}\).
    5. \(f(x) = 3x^4 + 4x^2 + 5\)
      \(f'(x)=12x^3+8x\).
  2. Let \(f(x)=x^2+x\).
    1. Find \(f'(x)\).
      \(f'(x)=2x+1\).
    2. Use the formula from the first part to find \(f'(2)\), \(f'(5)\).
      \(f'(2)=5\); \(f'(5)=11\).
  3. Let \(f(x)=2x^3+2x\).
    1. Find \(f'(x)\).
      \(f'(x)=6x^2+2\).
    2. Use the formula from the first part to find \(f'(1)\), \(f'(3)\).
      \(f'(1)=8\);\(f'(3)=56\).
  4. Let \(f(x) = 3x^4 + 7x^2 -6\).
    1. Find \(f'(x)\).
      \(f'(x)=12x^3+14x\).
    2. Find \(f''(x)\).
      \(f''(x)=36x^2+14\).
  5. Let \(f(x)=3x^2-x\). Find an equation for the tangent line to the graph \(y=f(x)\) at \(x=1\).
    The tangent line is given by \(y-2=5(x-1)\).
  6. Let \(f(x) = 2x^3 + 3x^2 + x\).
    1. Find \(f'(x)\).
      \(f'(x) = 6x^{2}+6x+1\)
    2. Find \(f''(x)\).
      \(f''(x)=12x+6\)
    3. Find an equation for the tangent line to the graph of \(f\) at \(x=2\).
      The tangent line is given by \(y-30=37(x-2)\).
  7. Let \(f(x)=2x^2 -x +1\).
    1. Find the tangent line to the graph of \(f\) at \(x=2\).
      An equation of the tangent line is \(y-7=7(x-2)\).
    2. Use the tangent approximation (about \(x=2\)) to approximate \(f(2.1)\).
      \(f(2.1)\approx7.7\).
  8. Let \(f(x)=x^2 +3x -1\).
    1. Find \(f(3)\).
      \(f(3)=17\).
    2. Find \(f'(3)\).
      \(f'(3)=9\).
    3. Use the tangent approximation (about \(x=3\)) to approximate \(f(3.1)\).
      \(f(3.1)\approx 17.9\).
    4. Find \(f(3.1)\) exactly.
      \(f(3.1)= 17.91\).
