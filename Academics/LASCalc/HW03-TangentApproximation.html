LAS Calculus
The Tangent Approximation
Homework 3
  1. For the next 1/2 hour, a car travels at an average velocity of 40 miles/hour. If it is currently 100 miles away and moving away, how far away will it be in 1/2 hour?
    The car will be \(120\) miles away.
  2. A car currently has an instantaneous velocity of 40 miles per hour.
    1. If it is currently 10 miles away and moving away, about how far will it be in 5 minutes?
      It will be about \(13 \frac{1}{3}\) miles away.
    2. Why can’t we get a good estimate of how far it travels in the next hour?
      Because 1 hour is too long a time; we can only use the instantaneous velocity to approximate average velocities for short time periods. Since we don’t know how fast it is going for that time, we can’t approximate how far it travels.
  3. A function \(y=f(x)\) satisfies \(f'(5)=3\). If \(f(5) = 10\), what is the approximate values of \(f(5.2)\)?
    \(f(5.2)\approx 10.6\).
  4. A function \(y=f(x)\) satisfies \(f(3) = 10\) and \(f'(3) = 4\). Approximately what is \(f(3.25)\)?
    \(f(3.25)\approx 11\).
  5. A function \(f(x)\) satisfies \(f(10) = 50\) and \(f'(10) = 2\). Approximate:
    1. \(f(11)\)
      \(f(11)\approx 52\).
    2. \(f(10.5)\)
      \(f(10.5)\approx 51\).
    3. \(f(9.5)\)
      \(f(9.5)\approx 49\).
  6. A function \(f(x)\) satisfies \(f(7) = 10\) and \(f'(7) = -2\). Approximate:
    1. \(f(8)\)
      \(f(8)\approx 8\).
    2. \(f(7.5)\)
      \(f(7.5)\approx 9\).
    3. \(f(6.8)\)
      \(f(6.8)\approx 10.4\).
  7. A function \(f(x)\) satisfies \(f(2) = 10\).
    1. If \(f'(2)=3\), use the tangent approximation to approximate \(f(2.5)\).
      \(f(2.5)\approx 11.5\).
    2. If \(f'(2.5)=3.5\), use your answer to (a) and the tangent approximation centered at \(2.5\) to approximate \(f(3)\).
      \(f(3)\approx 13.25\).
  8. A function \(f(x)\) satisfies \(f(5) = 12\).
    1. If \(f'(5)=2\), use the tangent approximation to approximate \(f(5.1)\).
      \(f(5.1)\approx 12.2\).
    2. If \(f'(5.1)=1.5\), use your answer to (a) and the tangent approximation centered at \(5.1\) to approximate \(f(5.2)\).
      \(f(5.2)\approx 12.35\).
