Section 1
\(\mathbb{R}^{n}\)

If you’ve had Calculus, you may have come across vectors and \(\mathbb{R}^{n}\), but even so a brief reminder may be helpful.

0.1  Definition of \(\mathbb{R}^{n}\)


Definition
An element of \(\mathbb{R}^{n}\) is an ordered list of \(n\) real numbers.

We will write the ordered lists as columns. So, for example, \(\mathchoice{\begin{pmatrix} 1\\2 \end{pmatrix}}{\small \begin{pmatrix} 1\\2 \end{pmatrix}}{}{}\) is an element of \(\mathbb{R}^{2}\), and \(\mathchoice{\begin{pmatrix} 3\\2\\5 \end{pmatrix}}{\small \begin{pmatrix} 3\\2\\5 \end{pmatrix}}{}{}\) is an element of \(\mathbb{R}^{3}\). If you learned about \(\mathbb{R}^{n}\) in Calculus, the elements were probably given as rows rather than columns; an element of \(\mathbb{R}^{2}\) would be written \((x,y)\) rather than \(\mathchoice{\begin{pmatrix} x\\y \end{pmatrix}}{\small \begin{pmatrix} x\\y \end{pmatrix}}{}{}\). While you should be familiar with both ways of representing elements of \(\mathbb{R}^{n}\), when dealing with matrices it is usually more useful to think of elements of \(\mathbb{R}{n}\) as columns.

An element of \(\mathbb{R}^{n}\) is often represented by \(\vect{x} = \begin{pmatrix} x_{1} \\ x_{2} \\ \vdots \\ x_{n}\end{pmatrix}\), although, to avoid subscripts, elements of \(\mathbb{R}^{2}\) are usually represented as \(\mathchoice{\begin{pmatrix} x\\y \end{pmatrix}}{\small \begin{pmatrix} x\\y \end{pmatrix}}{}{}\), and elements of \(\mathbb{R}^{3}\) by \(\mathchoice{\begin{pmatrix} x\\y\\z \end{pmatrix}}{\small \begin{pmatrix} x\\y\\z \end{pmatrix}}{}{}\). The numbers \(x_1\), \(x_2\), …, \(x_n\) (or \(x\), \(y\), \(z\)) are called the coordinates.

0.2  Points

The set \(\mathbb{R}^{1}\) is simply the set of real numbers, which is viewed geometrically as a line. The line is then called the real line. To make the identification between \(\mathbb{R}^{1}\) and a line, we need a special point on the line, called the origin, a unit length, and a positive direction. (Normally, the line is drawn horizontally and the positive direction is to the right.) Any real number \(x\) is associated with the following point on the line (see figure

  • If \(x = 0\), then \(x\) is associated with the origin.
  • If \(x > 0\), then \(x\) is associated with the point which is \(x\) units from the origin in the positive direction.
  • If \(x < 0\), then \(x\) is associated with the point which is \(|x|\) units from the origin in the negative direction (i.e., the opposite direction from the positive direction).

The set \(\mathbb{R}^{2}\) can be viewed geometrically as a plane. This identification needs two copies of the real line which intersect at right angles at their origins. One copy of the real line is the \(x\)-axis, one is the \(y\)-axis, and they are oriented so that going about the origin from the positive \(x\)-axis to the positive \(y\)-axis is counterclockwise. (This is called the positive orientation.) The \(x\)-axis is usually drawn horizontally with the positive \(x\) direction to the right, meaning the \(y\)-axis is vertical and the positive \(y\) direction is up. (See figure ??.)

Figure 1: The \(x\)- and \(y\)-axes.

An element \(\mathchoice{\begin{pmatrix} x\\y \end{pmatrix}}{\small \begin{pmatrix} x\\y \end{pmatrix}}{}{}\) in \(\mathbb{R}^{2}\) is associated with the point in the plane which is \(x\) units from the origin in the \(x\) direction and \(y\) units from the origin in the \(y\) direction. (See figure ??.)

Figure 2: An element of \(\mathbb{R}^{2}\) is associated with a point in the plane.

Continuing, the set \(\mathbb{R}^{3}\) can be viewed geometrically as space. This identification needs three copies of the real line which intersect at right angles at their origins. One copy of the real line is the \(x\)-axis, one is the \(y\)-axis, and one is the \(z\)-axis. They are oriented according to the right-hand rule, namely if you point the fingers on your right hand in the positive \(x\) direction so they bend in the positive \(y\) direction, your thumb will point in the positive \(z\) direction. (This is called the positive orientation.) The \(x\)-axis is usually indicated so that it is coming out the page, the \(y\)-axis is going to the right and the \(z\) axis is going up. (See figure ??.)

Figure 3: The \(x\), \(y\) and \(z\) axes in space; the \(x\) axis is supposed to be coming out of the page.

An element \(\mathchoice{\begin{pmatrix} x\\y\\z \end{pmatrix}}{\small \begin{pmatrix} x\\y\\z \end{pmatrix}}{}{}\) in \(\mathbb{R}^{3}\) is associated with the point in space which is \(x\) units from the origin in the \(x\) direction, \(y\) units from the origin in the \(y\) direction and \(z\) units from the origin in the \(z\) direction. (See figure ??.)

Figure 4: An element of \(\mathbb{R}^{3}\) is associated with a point in space.

This geometric association can be continued, but for \(n > 3\), \(\mathbb{R}^{n}\) becomes hard to visualize and even harder to draw. Because of this, when pictures are helpful, they will typically be in \(\mathbb{R}^{2}\).

0.3  Vectors

The coordinates of an element of \(\mathbb{R}^{n}\) can be thought of as directions to the point, starting at the origin. To get to the point in the plane given by \(\mathchoice{\begin{pmatrix} 2\\3 \end{pmatrix}}{\small \begin{pmatrix} 2\\3 \end{pmatrix}}{}{}\) from the origin, go \(2\) units in the \(x\) direction and \(3\) units in the \(y\) direction. These directions make sense even if we don’t start at the origin; starting at the point given by \(\mathchoice{\begin{pmatrix} 5\\7 \end{pmatrix}}{\small \begin{pmatrix} 5\\7 \end{pmatrix}}{}{}\), after going \(2\) units in the \(x\) direction and \(3\) units in the \(y\) direction, we will end up at the point given by \(\vectr{5+2\\ 7+3} = \vectr{7\\ 10}\).

Viewed as directions, an element of \(\mathbb{R}^{n}\) is called a vector. Vectors are often represented by an arrow from the origin to the point. Any arrows with the same direction and length represent the same vector (see figure ??), and so they don’t need to begin at the origin.

Figure 5: Both arrows represent the same vector in \(\mathbb{R}^{n}\).

When written by hand, a vector is typically written with an arrow over it, as \(\vec{x}\), but when typeset they are usually written in boldface, as \(\mathbf{x}\).

The vector from \(\vect{x} = \vectr{x_{1} \\ x_{2} \\ \vdots \\ x_{n}}\) to \(\vect{y} = \vectr{y_{1} \\ y_{2} \\ \vdots \\ y_{n}}\) is given by \(\vect{v} = \vectr{y_1 - x_{1} \\ y_2 - x_{2} \\ \vdots \\ y_n - x_{n}}\), abbreviated \(\vect{v} = \vect{y} - \vect{x}\). For example, in \(\mathbb{R}^{2}\), the vector from \(\mathchoice{\begin{pmatrix} 4\\2 \end{pmatrix}}{\small \begin{pmatrix} 4\\2 \end{pmatrix}}{}{}\) to \(\mathchoice{\begin{pmatrix} 3\\4 \end{pmatrix}}{\small \begin{pmatrix} 3\\4 \end{pmatrix}}{}{}\) is \(\vect{v} = \vectr{3 - 4\\4-2} = \vectr{-1\\2}\) (see figure ??).

Figure 6: The vector between two points.

0.4  Vector operations

Two elements of \(\mathbb{R}^{n}\) can be added by adding the corresponding elements.


Example
The sum of the two elements \(\mathchoice{\begin{pmatrix} 2\\3\\5 \end{pmatrix}}{\small \begin{pmatrix} 2\\3\\5 \end{pmatrix}}{}{}\) and \(\mathchoice{\begin{pmatrix} 3\\1\\7 \end{pmatrix}}{\small \begin{pmatrix} 3\\1\\7 \end{pmatrix}}{}{}\) in \(\mathbb{R}^{3}\) is \(\vectr{2\\3\\5}+\vectr{3\\1\\7}=\vectr{2+3\\3+1\\5+7}=\vectr{5\\4\\12}\).

The sum \(\vect{x}+\vect{y}\) of two vectors in \(\mathbb{R}^{n}\) can be thought of as following the directions given by \(\vect{x}\) and then following the directions given by \(\vect{y}\). Geometrically, to add \(\vect{x}+\vect{y}\), take the tail of an arrow representing \(\vect{y}\) and place it on the tip of an arrow representing \(\vect{x}\) (see figure ??).

Figure 7: Placing vectors in \(\mathbb{R}^{n}\) tip to tail.

The arrow representing \(\vect{x}+\vect{y}\) is then the arrow from the tail of the \(\vect{x}\) arrow to the tip of the \(\vect{y}\) arrow (see figure ??).

Figure 8: Adding two vectors in \(\mathbb{R}^{n}\).

Notice that \(\mathbb{R}^{n}\) has the special element \(\vect{0}=\vectr{0\\ \vdots\\0}\) which is the additive identity: \(\vect{v}+\vect{0} = \vect{v}\) for any \(\vect{v} \in \mathbb{R}^{n}\).

Vectors can also be multiplied by numbers; this is done by multiplying each element of the vector by the number.


Example
The product of \(5\) and the element \(\mathchoice{\begin{pmatrix} 2\\1\\2\\3 \end{pmatrix}}{\small \begin{pmatrix} 2\\1\\2\\3 \end{pmatrix}}{}{}\) of \(\mathbb{R}^{4}\) is \(\vectr{5\cdot 2\\ 5\cdot1\\ 5\cdot2\\ 5\cdot3}=\vectr{10\\5\\10\\15}\).

Multiplying a vector \(\vect{x}\) by a number \(c\) can be thought of as following the directions given by \(\vect{x}\), but going \(c\) times as far. Geometrically, multiplying \(\vect{x}\) by \(c\) results in an arrow with the same direction as the arrow representing \(\vect{x}\) (or reversing the direction, if \(c\) is negative), but changing the length by a factor of \(c\) (see figure ??).

Figure 9: Multiplying a vector by a number.

Since multiplying a vector by a number serves to scale the length of the vector, this type of multiplication is called scalar multiplication, and numbers are called scalars.

Addition and scalar multiplication of vectors satisfy the following properties.

Proposition Properties of addition and scalar multiplication

Commutivity of addition.
For all vectors \(\mathbf{x}\) and \(\mathbf{y}\), we have \(\vect{x} + \vect{y} = \vect{y} + \vect{x}\).
Associativity of addition.
For all vectors \(\mathbf{x}\), \(\mathbf{y}\) and \(\mathbf{z}\), we have \((\vect{x} + \vect{y}) + \vect{z} = \vect{x} + (\vect{y} + \vect{z})\).
Additive identity.
For the vector \(\mathbf{0}\), we have \(\vect{x} + \vect{0} = \vect{x}\) for all vectors \(\mathbf{x}\).
Additive inverses.
For any vector \(\mathbf{x}\), there is a vector \(-\vect{x}\) such that \(\vect{x} + -\vect{x} = \vect{0}\).
Associativity of multiplication.
For any vector \(\mathbf{x}\) and scalars \(a\) and \(b\), we have \(a(b\vect{x}) = (a b)\vect{x}\).
Multiplicative identity.
For any vector \(\mathbf{x}\), we have \(1 \vect{x} = \vect{x}\).
Distributivity I.
For any scalar \(a\) and vectors \(\mathbf{x}\) and \(\mathbf{y}\), we have \(a (\vect{x} + \vect{y}) = a \vect{x} + a \vect{y}\).
Distributivity II.
For any scalars \(a\) and \(b\) and vector \(\mathbf{x}\),we have \((a + b)\vect{x} = a \vect{x} + b \vect{x}\).

Addition and scalar multiplication are the two vector space operations. Although we will (mostly) restrict our attention to \(\mathbb{R}^{n}\), any set in which there is defined addition and scalar multiplication is called a vector space.


Definition
A vector space is a set \(V\) on which there are two operations:

  1. Addition: if \(\vect{x},\vect{y} \in V\), then \(\vect{x}+\vect{y} \in V\).
  2. Scalar multiplication: if \(\vect{x}\in V\) and \(c \in \mathbb{R}\), then \(c\vect{x} \in V\).

These operations must satisfy the properties of addition and scalar multiplication listed above.

An example of a vector space (besides \(\mathbb{R}^{n}\)) is the set of all polynomials. Any two polynomials can be added to get another polynomial, and a polynomial can be multiplied by a number to get another polynomial. For any fixed positive integer \(n\), the set \(\mathcal{P}_{n}\) of polynomials of degree \(n\) or less — the set of polynomials of the form \(p(x) = a_{n}x^{n} + \dots + a_{0}\) — is also a vector space. What’s more, the set \(\mathcal{P}_{n}\) can be identified with \(\mathbb{R}^{n+1}\); the element \(\vectr{a_{n} \\ a_{n-1} \\ \vdots \\ a_{0}}\) of \(\mathbb{R}^{n+1}\) can be considered an alternate way of writing the polynomial \(p(x) = a_{n}x^{n} + \dots + a_{0}\). Under this association, addition and scalar multiplication behave properly. For example, adding two polynomials in \(\mathcal{P}_{2}\), \((a_{1}x^{2}+b_{1}x+c_{1})+(a_{2}x^{2}+b_{2}x+c_{2}) =(a_{1}+a_{2})x^{2}+(b_{1}+b_{2})x+(c_{1}+c_{2})\), corresponds to addition in \(\mathbb{R}^{3}\), \((a_{1} \quad b_{1} \quad c_{1})^{T} + (a_{2} \quad b_{2} \quad c_{2})^{T} = (a_{1}+a_{2} \quad b_{1} + b_{2} \quad c_{1}+c_{2})^{T}\). Similarly, scalar multiplication in \(\mathcal{P}_{2}\) corresponds to scalar multiplication in \(\mathbb{R}^{3}\).

0.5  Length and direction of vectors

Vectors in \(\mathbb{R}^{n}\) can be used to indicate distance and direction. The distance indicated by a vector is called its length. Using the Pythagorean Theorem, we can show that the distance from the tail to the tip of a vector \(\vect{x} = \vectr{x\\y}\) in \(\mathbb{R}^{2}\) is \(\sqrt{x^2 + y^2}\). (See figure ??.)

Figure 10: The length of \(\mathbf{x}\).


Definition
The length of a vector \[\vect{x} = \vectr{x_1 \\ x_2 \\ \vdots \\ x_n}\] in \(\mathbb{R}^{n}\) is \[ \|\vect{x}\| = \sqrt{x_1^2 + x_2^2 + \dots + x_n^2}.\]


Example
Let \(\vect{x} = \vectr{2\\1\\3}\). Then \(\vect{x} = \sqrt{2^2 + 1^2 + 3^2} = \sqrt{14}\).

Related to the length of a vector is the dot product. (Since the result of the dot product is a scalar, this is also called the scalar product.)


Definition
The dot product (or scalar product) of two vectors \[ \vect{x} = \vectr{x_1\\x_2\\ \vdots \\ x_n} \text{ and } \vect{y} = \vectr{y_1\\ y_2 \\ \vdots \\ y_n}\] is \[\vect{x}\cdot \vect{y} = x_1 y_1 + x_2 y_2 + \dots + x_n y_n.\]


Example
Let \[\vect{x} = \vectr{1\\2\\1}, \vect{y}=\vectr{3\\2\\1} \in \mathbb{R}^{n}.\] Then \[ \vect{x}\cdot\vect{y} = 1\cdot 3 + 2\cdot 2 + 1\cdot 1 = 8.\]

Note that for any \(\vect{x}\in\mathbb{R}^{n}\), \(\|\vect{x}\| = \sqrt{\vect{x}\cdot\vect{x}}\).

The dot product has a geometric interpretation. If \(\mathbf{x}\) and \(\mathbf{y}\) are two vectors in \(\mathbb{R}^{n}\), let \(\theta\) be the angle between them. (See figure ??.)

Figure 11: The angle between two vectors.

Then \[ \vect{x}\cdot\vect{y} = \|\vect{x}\| \|\vect{y}\| \cos(\theta).\] Note that when the angle is \(\theta = 90^\circ\), then \(\cos(\theta) = 0\) and so \(\vect{x}\cdot\vect{y} = 0\). This is a useful characterization of what it means for two vectors to be perpendicular.


Remark
Two vectors \(\vect{x},\vect{y}\in\mathbb{R}^{n}\) are orthogonal (or perpendicular) if \(\vect{x}\cdot\vect{y} = 0\)


Example
The vectors \(\vect{x} = \vectr{1\\2}\) and \(\vect{y} = \vectr{-2\\1}\) in \(\mathbb{R}^{2}\) are orthogonal, since \(\vect{x}\cdot\vect{y} = 1\cdot (-2) + 2\cdot 1 = 0\). (See figure ??.)

Figure 12: Orthogonal vectors.

The dot product satisfies the following properties.

Proposition Properties of the dot product

Commutivity.
For all vectors \(\mathbf{x}\) and \(\mathbf{y}\), we have \(\vect{x} \cdot \vect{y} = \vect{y} \cdot \vect{x}\).
Distributivity.
For any scalar \(a\) and vectors \(\mathbf{x}\) and \(\mathbf{y}\), we have \(a (\vect{x} \cdot \vect{y}) = (a \vect{x}) \cdot \vect{y} = \vect{x}\cdot (a \vect{y})\).
Distributivity II.
For any vector \(\mathbf{x}\), we have \(\vect{x}\cdot\vect{x} \ge 0\), and \(\vect{x}\cdot\vect{x} = 0\) exactly when \(\vect{x} = \vect{0}\).

Any vector space with an operation which satisfies these properties is called an inner product space, since the dot product is sometimes called the inner product. (We are just using real numbers; if we were using complex numbers the definition of the inner product would be slightly different, but we won’t worry about that.)

0.6  Subspaces of \(\mathbb{R}^{n}\)

Even when we restrict our attention to \(\mathbb{R}^{n}\), we will come across more vector spaces than \(\mathbb{R}^{n}\) itself. A subset of \(\mathbb{R}^{n}\) will have a special status if addition and scalar multiplication will preserve it. Specifically:

Definition
A non-empty subset \(S\) of a vector space is a subspace if:

  1. For any \(\vect{x}, \vect{y} \in S\), their sum \(\vect{x}+\vect{y}\) is also in \(S\). (We say that \(S\) is closed under addition.)
  2. For any \(\vect{x} \in S\), \(c \in \mathbb{R}\), the scalar product \(c\vect{x}\) is also in \(S\). (We say that \(S\) is closed under scalar multiplication.)

Every \(\mathbb{R}^{n}\) has two somewhat trivial subspaces. The entire space \(\mathbb{R}^{n}\) is a subspace of itself, and the one element subset \(\left\{\vect{0}\right\}\) is also a subspace of \(\mathbb{R}^{n}\). Every other subspace is between these two: If \(S\) is a subspace of \(\mathbb{R}^{n}\), then by definition \(S\) must be contained in \(\mathbb{R}^{n}\), but also the element \(\vect{0}\) must be an element of \(S\).


Example
The set of vectors in \(\mathbb{R}^{3}\) whose second component is \(0\) (i.e., elements of the form \(\vectr{a\\0\\b}\)) is a subspace of \(\mathbb{R}^{3}\). This is straightforward to check:

  1. We can add two elements of this type \[\vectr{a_{1}\\0\\b_{1}} + \vectr{a_{2}\\0\\b_{2}} = \vectr{a_{1}+a_{2}\\0\\b_{1}+b_{2}}\] to get another element of this type. So this set is closed under addition.
  2. We can multiply an element of this type by a number \[c\vectr{a\\0\\b} = \vectr{ca\\0\\cb}\] to get another element of this type. So this set is closed under scalar multiplication.

So this is indeed a subspace.


Example
The set of vectors in \(\mathbb{R}^{4}\) whose first element is \(1\) (i.e., elements of the form \(\vectr{1\\a\\b\\c}\)) is not a subspace of \(\mathbb{R}^{4}\), since the zero element \(\vectr{0}=\vectr{0\\0\\0\\0}\) is not in this set.


Example
Let \(\mathcal{S}\) be the set of vectors in \(\mathbb{R}^{3}\) in which the first or second component is \(0\) (i.e., the set of vectors of the form \(\vectr{a\\b\\c}\) where either \(a=0\) or \(b=0\)).

This set \(\mathcal{S}\) is not a subspace of \(\mathbb{R}^{3}\). Note that \(\mathcal{S}\) contains the zero vector, but that is not enough to ensure the set is a subspace. In this case, we can add two elements of this set and get an element not in this set: \[\vectr{0\\1\\1}+\vectr{1\\0\\0}=\vectr{1\\1\\1},\] so the set is not closed under addition, and so is not a subspace of \(\mathbb{R}^{3}\).

We will later discuss a convenient way of characterizing subspaces of \(\mathbb{R}^{n}\).

If we want to visualize the subspaces of \(\mathbb{R}^{n}\), let’s start with \(\mathbb{R}^{2}\).

  1. The trivial subspace \(\{\vect{0}\}\) is itself a subspace.
  2. If a subspace contains \(\vect{x} \not=0\), then it must contain all multiples of \(\vect{x}\), \(\{c\vect{x}: c \in \mathbb{R}\}\). The set \(\vect{x}\), \(\{c\vect{x}: c \in \mathbb{R}\}\) is itself a subspace of \(\mathbb{R}^{2}\), corresponding to a line through the origin.
  3. If a subspace contains \(\vect{x} \not=0\) and \(\vect{y}\) which is not a multiple of \(\vect{x}\), then it will have to be all of \(\mathbb{R}^{2}\).

So, geometrically, the subspaces of \(\mathbb{R}^{2}\) are:

  1. The trivial subspace \(\left\{\vect{0}\right\}=\left\{\vectr{0\\0}\right\}\)
  2. Lines through the origin
  3. \(\mathbb{R}^{2}\) itself.

Similarly, the subspaces of \(\mathbb{R}^{3}\) are:

  1. The trivial subspace \(\left\{\vect{0}\right\}=\left\{\vectr{0\\0\\0}\right\}\)
  2. Lines through the origin.
  3. Planes through the origin.
  4. \(\mathbb{R}^{3}\) itself.

Xcas

Xcas/giac (https://www-fourier.ujf-grenoble.fr/~parisse/giac.html) is a computer algebra system freely available and easily installable on most computer operating systems and smartphones. (Windows, Mac OSX, Linux, Android, iOS, for example.) What’s more, it can be run inside a javascript enabled browser (it works well with Firefox).

Computers and smartphones are not usually allowed on tests, but Xcas is also available on some calculators. It is the built-in computer algebra system for the HP Prime, and can be installed on some other calculators, such as the TI Nspire calculator and the lower end and less expensive Casio fx-CG50 (see https://www-fourier.ujf-grenoble.fr/~parisse/casio/khicasioen.html), which is roughly equivalent to the TI-84+.

These notes will discuss how to use Xcas to do the matrix operations that we discuss in class. To find more about the capabilities of Xcas, see the manual (https://www-fourier.ujf-grenoble.fr/~parisse/giac.html#doc).

XCas can function as a simple calculator. If you start the program, you will be met with a prompt 

>>

You can evaluate a mathematical expression by typing it in and hitting enter.

You can enter more than one expression on the same line by separating any expressions with semicolons; the results will be separated by commas.

Giac has all of the usual operators (+, -, *, /,^, etc.), functions (sin, cos, tan, exp, ln, etc.) and constants (e, pi, etc.).



Xcas will give you exact results when possible. If the input is approximate (as indicated by a number with a decimal point), you will get a decimal approximation.



You can also ensure a decimal approximation by using the evalf function.

The evalf function takes a second argument indicating the desired precision. For example, you can get a decimal approximation for \(\pi\):

or you can get a decimal approximation with 50 digits to \(\pi\):

(The default number of decimals for evalf is configurable; see the manual for more details.)

You can assign values to variable names with the := operator.



You can create a list of objects (such as numbers, variables or expressions) by putting them between square brackets, separated by commas.

The number of elements in a list is given by the size command.

If you have a list, you can extract an element from the list by following the list with the index within square brackets. Indexing begins at 0, so to get the first element of the list, you can enter it followed with [0].





Indexing typically begins with 0 in computer programs, but normally in mathematics it begins with 1. You can use indices beginning with 1 in Xcas by enclosing the index in double square brackets, meaning that you can get the first element of a list by following it with [[1]].





A row vector can be represented by a list of numbers. For reasons that will come up later, to represent a column vector, you use a list where each number is in square brackets.



To find the length of a vector, you can use the norm command. (The length command will give you the number of elements in the list. The length of a vector is also called the norm, so the norm command returns the length/norm.)







The dot command will find the dot product of two vectors.



To find the dot product of two row vectors, you can also simply multiply them.

Vectors can be added and multiplied by scalars using the ordinary addition and multiplication operators.
