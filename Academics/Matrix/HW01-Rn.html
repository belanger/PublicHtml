Matrix Algebra
\(\mathbb{R}{n}\)
Homework 1
  1. For each pair of points \(\mathbf{x}\) and \(\mathbf{y}\), draw \(\mathbf{x}\), \(\mathbf{y}\) and the vector from \(\mathbf{x}\) to \(\mathbf{y}\). Find the vector from \(\mathbf{x}\) to \(\mathbf{y}\), and use it to find the distance from \(\mathbf{x}\) to \(\mathbf{y}\).
    1. \(\vect{x} = \vectr{1\\2}, \vect{y} = \vectr{2\\1}\)
      The vector from \(\mathbf{x}\) to \(\mathbf{y}\) is \(\vect{y}-\vect{x} = \vectr{1\\-1}\). The distance from \(\mathbf{x}\) to \(\mathbf{y}\) is \(\sqrt{2}\).
    2. \(\vect{x} = \vectr{-2\\1}, \vect{y} = \vectr{3\\-1}\)
      The vector from \(\mathbf{x}\) to \(\mathbf{y}\) is \(\vect{y}-\vect{x} = \vectr{5\\-2}\). The distance from \(\mathbf{x}\) to \(\mathbf{y}\) is \(\sqrt{29}\).
    3. \(\vect{x} = \vectr{-1\\-1}, \vect{y} = \vectr{2\\3}\)
      The vector from \(\mathbf{x}\) to \(\mathbf{y}\) is \(\vect{y}-\vect{x} = \vectr{3\\4}\). The distance from \(\mathbf{x}\) to \(\mathbf{y}\) is \(5\).
    4. \(\vect{x} = \vectr{-2\\3}, \vect{y} = \vectr{4\\-2}\)
      The vector from \(\mathbf{x}\) to \(\mathbf{y}\) is \(\vect{y}-\vect{x} = \vectr{6\\-5}\). The distance from \(\mathbf{x}\) to \(\mathbf{y}\) is \(\sqrt{61}\).
  2. For each set of vectors \(\mathbf{x_1}\), \(\mathbf{y_1}\), \(\mathbf{x_2}\) and \(\mathbf{y_2}\), determine whether or not the vector from \(\mathbf{x_1}\) to \(\mathbf{y_1}\) is equal to the vector from \(\mathbf{x_2}\) to \(\mathbf{y_2}\).
    1. \(\vect{x_1} = \vectr{2\\1}, \vect{y_1} = \vectr{3\\4}, \vect{x_2} = \vectr{-1\\3}, \vect{y_2} = \vectr{0\\3}\)
      The vector from \(\mathbf{x_1}\) to \(\mathbf{y_1}\) is not equal to the vector from \(\mathbf{x_2}\) to \(\mathbf{y_2}\).
    2. \(\vect{x_1} = \vectr{1\\2}, \vect{y_1} = \vectr{-1\\-2}, \vect{x_2} = \vectr{3\\4}, \vect{y_2} = \vectr{5\\0}\)
      The vector from \(\mathbf{x_1}\) to \(\mathbf{y_1}\) is not equal to the vector from \(\mathbf{x_2}\) to \(\mathbf{y_2}\).
    3. \(\vect{x_1} = \vectr{1\\2\\3}, \vect{y_1} = \vectr{-3\\1\\7}, \vect{x_2} = \vectr{2\\5\\1}, \vect{y_2} = \vectr{-2\\4\\5}\)
      The vector from \(\mathbf{x_1}\) to \(\mathbf{y_1}\) is equal to the vector from \(\mathbf{x_2}\) to \(\mathbf{y_2}\).
    4. \(\vect{x_1} = \vectr{-1\\1\\-1}, \vect{y_1} = \vectr{2\\3\\4}, \vect{x_2} = \vectr{1\\5\\2}, \vect{y_2} = \vectr{-2\\3\\-3}\)
      The vector from \(\mathbf{x_1}\) to \(\mathbf{y_1}\) is not equal to the vector from \(\mathbf{x_2}\) to \(\mathbf{y_2}\).
  3. For each pair of vectors \(\mathbf{x}\) and \(\mathbf{y}\), find the dot product \(\vect{x}\cdot\vect{y}\). Determine whether or not \(\mathbf{x}\) and \(\mathbf{y}\) are orthogonal.
    1. \(\vect{x} = \vectr{1\\2}, \vect{y} = \vectr{2\\3}\)
      \(\vect{x}\cdot\vect{y} = 8\). The vectors are not orthogonal.
    2. \(\vect{x} = \vectr{-2\\3}, \vect{y} = \vectr{2\\-3}\)
      \(\vect{x}\cdot\vect{y} = -13\). The vectors are not orthogonal.
    3. \(\vect{x} = \vectr{1\\-2\\3}, \vect{y} = \vectr{1\\2\\1}\)
      \(\vect{x}\cdot\vect{y} = 0\). The vectors are orthogonal.
    4. \(\vect{x} = \vectr{2\\1\\-2}, \vect{y} = \vectr{1\\2\\1}\)
      \(\vect{x}\cdot\vect{y} = 2\). The vectors are not orthogonal.
  4. For each pair of vectors \(\mathbf{x}\) and \(\mathbf{y}\), find a value of \(c\) which makes \(\mathbf{x}\) and \(\mathbf{y}\) orthogonal.
    1. \(\vect{x} = \vectr{2\\3}, \vect{y} = \vectr{4\\c}\)
      \(c = -8/3\)
    2. \(\vect{x} = \vectr{-2\\2c}, \vect{y} = \vectr{3\\4}\)
      \(c = 3/4\)
    3. \(\vect{x} = \vectr{1\\2\\4}, \vect{y} = \vectr{3c\\2\\1}\)
      \(c = -8/3\)
    4. \(\vect{x} = \vectr{2\\1\\3c}, \vect{y} = \vectr{4c\\2\\-2}\)
      \(c = -1\)
