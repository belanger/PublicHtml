Emacs code

Comics
Comics is a program that will allow you to download and view comics.
Quotes mode
Quotes mode allows you to browse and search through a collection of quotes.
Quotes.gz is a collection that can be used.
Grade
The Grade program is a program to keep track of (and compute) students' grades.
Balance
The Balance mode is a modified version of the balance mode written by Bob Newell and Jason Baietto. I use it to keep track of my checking account. Credit card mode is a simplified version I use to keep track of my credit card bills.
Goobook mode
Goobook mode is an Emacs interface to goobook, which is a way to interact with Google Contacts.
